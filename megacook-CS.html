Mega Cook - Fullstack Development Decal Final Project

Project Overview

In Spring 2024, I took the Fullstack Development DeCal at UC Berkeley to learn the basics of HTML, CSS, and JavaScript on Visual Studio Code. I also learned how to design with Figma and make API calls.

Our final project was to create any website using API calls. My team decided to create a recipe generator website based on the use's ingredients.

Design Process

How might we create a website that can make cooking easier for busy people such as college students?
